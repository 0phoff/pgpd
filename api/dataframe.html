Geos DataFrame Accessor¶

class pgpd.GeosDataFrameAccessor(obj)¶

Access PyGEOS functionality through the “geos” dataframe accessor keyword. The functions defined here simply call the appropriate functions from GeosSeriesAccessor and group the results.

Example

>>> df = pd.DataFrame({
...     'a': list('abcde'),
...     'poly': pygeos.box(range(5), 0, range(10,15), 10),
...     'pt': pygeos.points(range(5), range(10,15))
... })
>>> df = df.astype({'poly':'geos', 'pt':'geos'})
>>> df
   a                                      poly            pt
0  a  POLYGON ((10 0, 10 10, 0 10, 0 0, 10 0))  POINT (0 10)
1  b  POLYGON ((11 0, 11 10, 1 10, 1 0, 11 0))  POINT (1 11)
2  c  POLYGON ((12 0, 12 10, 2 10, 2 0, 12 0))  POINT (2 12)
3  d  POLYGON ((13 0, 13 10, 3 10, 3 0, 13 0))  POINT (3 13)
4  e  POLYGON ((14 0, 14 10, 4 10, 4 0, 14 0))  POINT (4 14)
>>> df.geos.has_z()
    poly     pt
0  False  False
1  False  False
2  False  False
3  False  False
4  False  False
