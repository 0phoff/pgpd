apply¶

GeosSeriesAccessor.apply(*args, **kwargs)¶

Returns a copy of a geometry array with a function applied to its coordinates.

Applies pygeos.coordinates.apply to the data and returns a Series with the result.

Parameters

  • args – Arguments passed to apply after the first argument.

  • kwargs – Keyword arguments passed to apply.

Returns

Series with the result of the function and the same index.

Return type

pandas.Series
