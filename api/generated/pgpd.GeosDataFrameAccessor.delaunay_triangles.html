delaunay_triangles¶

GeosDataFrameAccessor.delaunay_triangles(*args, inplace=False, **kwargs)¶

Computes a Delaunay triangulation around the vertices of an input geometry.

Applies pgpd.GeosSeriesAccessor.delaunay_triangles() to each column of “geos” dtype and aggregates the results in a DataFrame.

Parameters

  • args – Arguments passed to delaunay_triangles() after the first argument.

  • inplace (bool, optional) – Whether to perform the modifications inplace; Default False.

  • kwargs – Keyword arguments passed to delaunay_triangles().

Returns

DataFrame where each “geos” column from the original is transformed or None if inplace=True.

Return type

pandas.DataFrame or None
