get_type_id¶

GeosDataFrameAccessor.get_type_id(*args, inplace=False, **kwargs)¶

Returns the type ID of a geometry.

Applies pgpd.GeosSeriesAccessor.get_type_id() to each column of “geos” dtype and aggregates the results in a DataFrame.

Parameters

  • args – Arguments passed to get_type_id() after the first argument.

  • inplace (bool, optional) – Whether to perform the modifications inplace; Default False.

  • kwargs – Keyword arguments passed to get_type_id().

Returns

DataFrame where each “geos” column from the original is transformed or None if inplace=True.

Return type

pandas.DataFrame or None
