length#

GeosDataFrameAccessor.length(*args, inplace=False, **kwargs)#

Computes the length of a (multi)linestring or polygon perimeter.

Applies pgpd.GeosSeriesAccessor.length() to each column of “geos” dtype and aggregates the results in a DataFrame.

Parameters:

  • args – Arguments passed to length() after the first argument.

  • inplace (bool, optional) – Whether to perform the modifications inplace; Default False.

  • kwargs – Keyword arguments passed to length().

Returns:

DataFrame where each “geos” column from the original is transformed or None if inplace=True.

Return type:

pandas.DataFrame or None
