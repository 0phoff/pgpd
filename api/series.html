Geos Series Accessor#

class pgpd.GeosSeriesAccessor(obj)#

Access shapely functionality through the “geos” series accessor keyword.

Example

>>> s = pd.Series(shapely.points(range(15), 0), dtype='geos')
>>> s
0    POINT (0 0)
1    POINT (1 0)
2    POINT (2 0)
3    POINT (3 0)
4    POINT (4 0)
5    POINT (5 0)
6    POINT (6 0)
7    POINT (7 0)
8    POINT (8 0)
9    POINT (9 0)
dtype: geos
>>> s.geos.has_z
0    False
1    False
2    False
3    False
4    False
5    False
6    False
7    False
8    False
9    False
Name: has_z, dtype: bool
